    仿淘宝电商系统

    前台：html + css + jQuery + AngularJs

    后台：Node.js + Express + MongoDB + Mongoose

    项目描述：

    前台实现完整电商流程所有页面，用jQuery实现页面特效以及跟后台数据交互，例如放大镜、购物车商品数量+- 等等。表单验证用了AngularJs。页面兼容Chrome、Firefox、IE7+、Opera 等浏览器。

    项目技术难点是购物车部分，仿淘宝商品选中/未选中状态用的是label不同背景图，这就造成了全/单/多选、选中店铺名全选店铺下所有商品选中/未选中/取消选中状态商品+- 改变结算值的复杂度，通过jQuery对代码不断更改，最终实现购物车功能。

    责任描述：

    独立完成从前台到后台开发。

    项目地址： https://github.com/JQLing/Online-retailers
    个人收获：

    深刻理解前后台数据交互、对 Cookie 和 Session 的运用和区别、加强对后台的运用。项目问题的出现使我分析问题更加准确，解决问题的方法更加有效。
